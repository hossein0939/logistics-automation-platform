# Logistics Automation Platform

A full-stack automation system designed to streamline logistics workflows.

## 🚀 Features
- Desktop automation tool built with C#
- Web-based admin panel (PHP)
- Backend services using .NET
- Android application for field users
- API integration and workflow automation

## 🧠 Overview
This project demonstrates building an end-to-end system including backend, frontend, and mobile components.

## 🛠 Tech Stack
- C#
- .NET
- PHP
- Android (Java/Kotlin)

## 📌 Highlights
- Automation of real-world workflows
- API integration
- Authentication handling (OTP-based systems)
